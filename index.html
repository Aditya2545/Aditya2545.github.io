// 05 — research

Research & Publications

Real-Time Digital Twin Synchronization via Collaborative Multi-Vehicle Visual SLAM
In Progress · Target: IEEE ICRA / IROS / RA-L · Texas A&M–Kingsville AI-Cyb Lab
Investigates how a fleet of Quanser QCar autonomous vehicles can collaboratively build a shared map (Digital Twin) in real time using Visual SLAM. Core contributions include Neural Map Fusion for per-vehicle submap merging, a Self-Supervised Divergence Detector for consistency monitoring, and a TensorRT-optimized feature extraction pipeline (32× speedup, sub-ms on Jetson Orin). Integrates SuperPoint/SuperGlue feature matching, GTSAM factor-graph optimization (LIO-SAM), and a 5-state C++17 EKF for multi-sensor fusion.
Camera–LiDAR Fusion for 3D Multi-Object Tracking (KITTI Benchmark)
Portfolio Project · ROS 2 · C++17
Complete camera-LiDAR fusion and 3D multi-object tracking pipeline on the KITTI benchmark. Includes ROS 2 fusion and tracker nodes written in C++17, with TensorRT inference benchmarked separately on Tesla T4. Demonstrates the full perception stack from raw sensor input through 3D bounding box estimation and track management.
6-DOF Robot Arm Kinematics — Pure C++17 Library
Open Source · github.com/Aditya2545/robot_arm_kinematics
Standalone kinematics library implementing Denavit-Hartenberg forward kinematics and damped-least-squares inverse kinematics for a 6-DOF manipulator — zero external dependencies. CMake static/shared builds, CLI tool, and complete Doxygen documentation, demonstrating the mathematical foundations underlying manipulator control.
